# HittaEvent

Find events from TicketMaster in Sweden. 

## Features

- Search events
- Browse events in chosen city
- Save events to favourites

## API

https://developer.ticketmaster.com/products-and-docs/apis/discovery-api/v2/

## Tech Stack
- TypeScript
- React Native & Expo

## API

https://developer.ticketmaster.com/products-and-docs/apis/discovery-api/v2/

## Getting started

1. Clone the project
   git clone <repo-url>

2. Install dependencies
   npm install

3. Create a .env file in the root and add your API key
   EXPO_PUBLIC_TICKETMASTER_API_KEY=your_key_here

4. Start the development server
   npx expo start

5. Scan the QR code with the Expo Go app
