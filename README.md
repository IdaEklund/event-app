# HittaEvent

Find events from TicketMaster in Sweden. 

## Status
🚧 Under development. Styling is not done. Some functionality will also be added.

## Features

- Search events
- Browse events in chosen city
- Save events to favourites

## Tech Stack
- TypeScript
- React Native & Expo
